# 💎 Projeto Solara Estética

O **Solara Estética** é uma solução inteligente para clínicas de estética focada em dois grandes gargalos: **Agendamento Operacional** e **Retenção de Receita**.

## 🎯 Objetivo Central
Transformar a gestão de clínicas de estética através de automação e inteligência artificial, garantindo que nenhum lead seja perdido e que clientes inativos sejam recuperados de forma proativa.

---

## 🏗️ Módulos do Sistema

### 📅 Gestão de Agendamentos
- **Visualização 360:** Controle de agenda por especialista e sala.
- **Integração:** Sincronização de status entre a recepção e o operacional.
- **Prevenção de No-show:** Sistema de confirmação automática.

### 🧠 Recuperação de Clientes (IA Solara)
- **Análise de Inatividade:** Identifica automaticamente quem não aparece há X dias.
- **Copywriting IA:** Gera mensagens personalizadas via Gemini AI para cada perfil de cliente.
- **Conversão Direct:** Link direto para re-agendamento via WhatsApp Live.

### 📈 Inteligência Comercial
- **Kanban de Leads:** Fluxo de vendas visual para conversão de novos clientes.
- **NPS:** Monitoramento da satisfação pós-procedimento.
- **Financeiro:** Controle de entradas, saídas e inadimplência.

---

## 🛠️ Arquitetura Técnica

- **Frontend:** Single Page Application (SPA) com React 19.
- **Backend:** API REST robusta com FastAPI.
- **Inteligência:** Integração nativa com `Google Generative AI` para suporte e recuperação.
- **UX/UI:** Design premium utilizando `Framer Motion` para transições suaves e `Tailwind` para consistência visual.
