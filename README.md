## 🚀 Sobre o Projeto
O **Solara Estética** é uma plataforma SaaS (Software as a Service) avançada, desenvolvida especificamente para clínicas de estética que buscam excelência operacional e crescimento acelerado através de Inteligência Artificial.

## 🛠️ Pilares Principais

### 1. 📅 Agendamento e Gestão de Agenda
- Controle total sobre consultas e procedimentos.
- Visão por especialista e disponibilidade em tempo real.
- Redução de faltas através de lembretes inteligentes.

### 2. ✨ Recuperação de Clientes com IA
- Sistema automatizado para identificar clientes inativos.
- Campanhas personalizadas de reativação via WhatsApp.
- Análise preditiva de churn (cancelamento).

### 3. 📊 Dashboard e Central Kanban
- Visualização clara do funil de vendas e leads.
- Indicadores de performance (KPIs) em tempo real.
- Gestão financeira integrada com controle de cobranças.

## 💻 Stack Tecnológica
- **Frontend:** React 19 + Vite + TypeScript
- **Estilização:** Tailwind CSS + Framer Motion (animações premium)
- **Estado:** Zustand
- **AI:** Google Gemini API integration
- **Ícones:** Lucide React
- **Backend:** FastAPI + PostgreSQL + Redis

## ⚙️ Instalação Local

**Pré-requisitos:** Node.js (v18+) e Docker (opcional para o backend completo)

1. **Clonar e instalar dependências do Frontend:**
   ```bash
   npm install
   ```

2. **Rodar o ambiente de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Backend (Docker):**
   ```bash
   cd backend
   docker-compose up -d
   ```
